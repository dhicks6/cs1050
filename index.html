Which political party thinks president trump was handeling the Covid 19 response the best

Democrats average approval rating from the data set was 21.1023%.

Democrats average dissapproval rating from the data set was 75.6880%.

Republicans average approval rating from the data set was 76.2964%.

Republicans average dissapproval rating from the data set was 20.4801%.

From the data we can see that democrate and republicans mirrored each other in how they felt the president handeled his covid 19 response.

With the averages we can say that republicans thought the president handled the reponse the best.

For this project I used data from https://data.fivethirtyeight.com/ which was one of the linked sources in the project home page.

The data was collected over a 6 month period in 2020. Many different sources are listed in the data.

To find the averages of the data I used a local mysql database, and loaded the provided .csv file into it. I then ran commands like

SELECT AVG(approve) FROM polls.covid_approval_polls WHERE party="d"; this command does is takes the approval percent from all polls

where democrats were the only ones polled. I then did the same for the democrat dissapproval rate, and then for republicans as well.

I have used Excel to make my graphs. They show the mirroring of how each party felt about the issue. At the end you will notice a sharp change.
